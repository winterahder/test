# 오상고 상담카드 시스템 v3.0 (Single-File)

## 업로드 방법
1. index.html을 GitHub의 최상위에 덮어쓰기
2. data/students.json, data/mock.json은 그대로 유지
3. 기존 js/, css/ 폴더는 삭제해도 되고 그대로 둬도 됨 (이번 버전은 사용 안함)

## 포함 기능
- 학생 목록 + 검색 + 학년/반 필터
- 7개 탭 (기본정보/모의고사/내신/세특/행발/수상/독서)
- 기본정보 한글 라벨 표 스타일
- 모의고사 그래프 Y축 정수 눈금
- 점수 토글 (등급/백분위/표점/원점수)
